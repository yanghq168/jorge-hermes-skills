## 📖 如何使用

每个分类目录中包含完整的 Agent 定义文件（`.md`），结构如下：

```yaml
---
name: Agent 名称
description: 职责描述
color: 主题色
emoji: 表情符号
vibe: 个性标签
---
```

内容包含：
- **🧠 角色定义** - 专业技能、个性特征、记忆系统
- **🎯 核心使命** - 职责范围、工作目标
- **⚙️ 工作流程** - 具体的执行步骤和方法
- **💾 记忆系统** - 上下文保持和经验积累
