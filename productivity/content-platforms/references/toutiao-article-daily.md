# 头条号文章生成器 toutiao-article-daily.py

## 脚本路径
`~/.hermes/cron/scripts/toutiao-article-daily.py`

## 定时任务
每天 20:30 运行

## 版本历史
- **v1.1.0**（当前）：发件人改为「权权的HERMES（头条）」，新增2条微头条/天，配图增至5张，增强钩子真实感细节
- **v1.0.5**（旧版）：深蓝Banner初版，发件人「围炉家常话（头条）」

## 与公众号版的区别

| 维度 | 公众号版 | 头条版 |
|------|---------|--------|
| 配色 | 紫色渐变 | 深蓝渐变 |
| 发件人 | 权权的HERMES（公众号） | 权权的HERMES（头条） |
| 选题方向 | 养老安全、金钱观、亲子关系 | 遗产分配、赡养义务、亲戚恩怨、房产纠纷、随礼人情 |
| 用户画像 | 中老年女性 | 中老年男性（45-70岁） |
| 文章风格 | 精致短文 | 长文吃流量，故事要完整 |
| 配图数量 | 3张 | 5张（封面1张+文中5张） |
| 微头条 | 无 | 2条/天，独立选题库 |

## 选题库特征（头条版）

- **遗产分配**：遗嘱、账本、数字说话
- **赡养义务**：轮流养老、据点vs旅馆
- **亲戚恩怨**：帮衬、规矩、人情账
- **房产纠纷**：过户、底牌、护身符
- **晚年孤独**：菜市场社交、定通话规矩
- **随礼人情**：撒网vs种树、假人情不如没有

## 微头条模块（v1.1.0新增）

独立于长文的微头条，每天生成2条：
- 字数：150-250字
- 选题：来自 `MICRO_TOPICS_POOL`（6个独立选题，每次随机抽2条）
- 结构：方向 → 标题 → 内容 → 对话高亮 → 配图提示词 → 标签
- 邮件格式：追加在长文HTML下方，用深蓝分隔线区分

### 微头条选题方向
- 亲戚帮衬：被道德绑架、断联
- 婆媳关系：距离产生美
- 养老金：算清楚再养老
- 子女不孝：一年5个电话要钱
- 老年婚姻：只认钱不认人
- 人情往来：假人情烧本子

## 核心数据结构

```python
topic = {
    "direction": "遗产分配",           # 选题方向
    "category": "情感·家庭婚姻",       # 头条类目
    "title": "72岁老人存了40万...",    # 文章标题（需含具体年龄/数字）
    "hook": "这是网友@老陈头 的投稿...",  # 钩子（需含真实感细节）
    "story_a": "...",                 # 故事A（负面/反例）
    "story_a_dialogue": "...",        # 故事A对话高亮
    "story_a_ending": "...",          # 故事A结尾
    "story_b": "...",                 # 故事B（正面/智慧案例）
    "story_b_ending": "...",          # 故事B结尾
    "viewpoint": "...",               # 反共识犀利观点
    "action": "...",                  # 给读者的行动台阶
    "golden1": "...",                 # 金句1（故事A结尾）
    "golden2": "...",                 # 金句2（故事B结尾）
    "golden3": "...",                 # 金句3（文章结尾）
    "tags": ["遗产分配", "遗嘱", ...], # 话题标签
    "cover_prompt": "... --ar 16:9 --v 6",  # 封面图DALL-E提示词
    "in_image_prompts": [             # 5张文中配图
        {"num": 1, "scene": "场景名", "desc": "描述", "prompt": "DALL-E提示词"},
        {"num": 2, "scene": "...", "desc": "...", "prompt": "..."},
        ...
    ],
}
```

### 钩子真实感要求（头条特色）
头条钩子不能是干巴巴一句话，必须包含：
- 具体时间："有一天晚上"、"那天凌晨"
- 具体地点："我家楼下那个菜市场"、"医院走廊里"
- 具体数字："38度5"、"15平米的出租屋"
- 具体对话：要有引号内的原话，不是"他说"

示例：
```
# ❌ 旧版（干巴巴）
"这是读者@老王 的真实经历，看完我心里很不是滋味。"

# ✅ v1.1.0（真实感细节）
"这是读者@老王 的投稿，讲的是他自己的故事。他说有一天他发烧，躺了一天，没人知道。晚上量体温，38度5，他给自己倒了一杯水，就这么睡着了。"
```

## 关键实现细节

1. **深蓝Banner**：`#1a2a4a → #162340 → #0f1a30` 渐变，稳重深沉
2. **金句高亮**：红色 `#c0392b`，对比公众号版更浓烈
3. **话题标签**：蓝色 `#2980b9`，与深蓝Banner呼应
4. **互动问题**：争议性，让两拨人想转发讨论
5. **配图提示词**：纪录片风格，情绪张力，无文字，16:9横版，--ar 16:9 --v 6
6. **钩子真实感**：头条钩子需包含具体时间、地点、数字、对话细节

- [ ] 有没有对话或心理描写？
