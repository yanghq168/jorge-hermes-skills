# 微信公众号「围炉家常话」输出样例

## 邮件标题
`【养老安全】70岁以上丧偶老人，千万别独居：两个邻居的血泪教训`

## 人设
**人间清醒观察家** — 不是主编/老师，是老读者身边爱讲故事的朋友
语气：讲述者、旁观者视角，不是教导者

## 文章结构（扩充版）

```
【钩子】150字
  "这是网友@李阿姨的投稿，看完我沉默了很久..."
  
【故事A】400字（负面/警示案例）
  王大妈，70岁，丧偶独居
  具体细节：时间、地点、人物、对话、数字
  结尾：一个让人心酸/寒心的结局

【金句1】独居不是自由，是风险。等出事了再说'我没事'，就晚了。

【故事B】300字（正面/对比案例）
  张大爷，68岁，同样情况，但应对方式不同
  结尾：子女轮流照顾，安享晚年

【金句2】对儿女'示弱'不是矫情，是老年人的生存智慧。

【人间清醒观点】200字
  提炼核心观点，不是说教，是讲故事
  
【行动指引】150字
  给出具体可操作的建议
  
【互动收尾】100字
  争议性问题 + 引导在看/转发
  
【金句3】存折上的钱，人没了就是废纸。活着，花了才是自己的。
```

## 四大内容方向
1. **养老安全** — 独居、存款、房产、子女不孝
2. **金钱观与健康** — 退休金、打拼vs享福、医疗费
3. **亲子关系与底线** — 房产过户、遗产分配、亲情绑架
4. **亲戚人情** — 随礼攀比、真心付出、边界感

## 选题库扩充版要点
- 选题要有对比性（故事A vs 故事B）
- 故事要有具体数字（退休金3400、存款25万、手术费6万）
- 人物要有姓名、年龄、职业
- 对话要真实可感
- 结局要有情感冲击力
